# <p align="center">MaomaoChanBot

### <p align="center">Bot resmi untuk moderasi grup Telegram

**MaomaoChanBot** adalah bot Telegram resmi buatan **Rizky** untuk membantu moderasi grup dengan efisien.  
Bot ini berbasis JavaScript dan MongoDB, serta dikembangkan dari proyek open-source.

### Instalasi
Clone Repository
```bash
git clone https://github.com/ikyYourStar/maomaoTele
```
Install Package
```bash
npm Install
```
Jalankan Bot
```bash
npm start
```
**Prasyarat:**  
- Node.js versi > 23  
- Bot Telegram aktif dan telah menonaktifkan *privacy mode*  
- Kanal pencatat (log channel)  
- Konfigurasi token bot, `chat_id`, dan `user_id` di `MaoStg.json`
