### Instalasi
Clone Repository
```bash
git clone https://github.com/ikyYourStar/maomaoTele
```
Install Package
```bash
npm Install
```
### SetUp Bot
Buka Telegram mu lalu ke botFather lalu buat bot mu setelah itu ambil token mu lalu ke code nya ke bsgian MaoStg.json paste token nya di token: ""
Jalankan Bot
```bash
npm start
```
**Prasyarat:**  
- Node.js versi > 23 
- Bot Telegram aktif dan telah menonaktifkan *privacy mode*  
- Kanal pencatat (log channel)  
- Konfigurasi token bot, `chat_id`, dan `user_id` di `MaoStg.json`
